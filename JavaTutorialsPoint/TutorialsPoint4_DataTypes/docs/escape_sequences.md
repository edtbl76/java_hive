# Java Supported Escape Sequences

    Notation    
    \n          newline                 0x0a
    \r          carriage return         0x0d
    \f          formfeed                0x0c
    \b          backspace               0x08
    \s          space                   0x20
    \t          tab
    \"          double quote
    \'          single quote
    \\          backslash
    \ddd        octal character         ddd
    \uxxxx      Hexadecimal Unicode     xxxxc
